# Screenshots Directory

This directory contains screenshots for the Google Photos to iCloud Migration Tool documentation.

## Required Screenshots

### Web UI Screenshots

Please add the following screenshots:

1. **`web-ui-dashboard.png`** - Main dashboard showing the web UI interface
   - Shows: Status badge, progress bar, statistics cards, configuration section

2. **`web-ui-running.png`** - Migration in progress
   - Shows: Running status, progress bar with percentage, live statistics updating, activity log

3. **`web-ui-completed.png`** - Completed migration
   - Shows: Completed status, final statistics, success message

4. **`web-ui-failed-uploads.png`** - Failed uploads view
   - Shows: Failed uploads list with file names and retry counts

### How to Add Screenshots

1. Take screenshots using your screenshot tool (Cmd+Shift+4 on macOS)
2. Save them in this directory with the names listed above
3. Screenshots should be:
   - PNG format
   - Reasonable size (1920x1080 or smaller)
   - Clear and readable

The documentation will automatically reference these screenshots once they're added.
